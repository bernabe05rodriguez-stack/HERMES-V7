GUÍA PARA COMPARTIR EL PROGRAMA "HERMES"
=========================================================

Esta guía explica cómo generar el archivo ejecutable (.exe) y cómo configurarlo para que tus amigos puedan usarlo conectándose a tu servidor.

---------------------------------------------------------
PASO 1: COMPILAR EL EJECUTABLE (.EXE)
---------------------------------------------------------
1. Abre la carpeta `client` dentro del proyecto.
2. Haz doble clic en el archivo `build_exe.bat`.
3. Se abrirá una ventana negra que ejecutará el proceso de compilación. Espera a que termine.
4. Al finalizar, verás una nueva carpeta llamada `dist`.
5. Dentro de `dist` encontrarás el archivo `HermesV7.exe`. Este es el programa principal.

---------------------------------------------------------
PASO 2: PREPARAR LOS ARCHIVOS PARA COMPARTIR
---------------------------------------------------------
Para que el programa funcione en la computadora de tu amigo, necesitas enviarle dos cosas:

1. El archivo `HermesV7.exe` (que acabas de crear).
2. El archivo `config.json` (que está en la carpeta `client`).

---------------------------------------------------------
PASO 3: CONFIGURAR LA DIRECCIÓN IP (IMPORTANTE)
---------------------------------------------------------
El archivo `config.json` le dice al programa dónde buscar tu servidor.

- Si tu amigo está en tu misma red WiFi:
  1. Averigua tu IP local (abre CMD y escribe `ipconfig`, busca "Dirección IPv4", ej: 192.168.1.X).
  2. Abre el archivo `config.json` con el Bloc de notas.
  3. Cambia "http://localhost:8000" por "http://TU_IP_LOCAL:8000" (ej: "http://192.168.1.15:8000").
  4. Guarda el archivo.

- Si tu amigo está en otra casa (Internet):
  1. Necesitas tu IP Pública (puedes verla en sitios como cual-es-mi-ip.net).
  2. Debes asegurarte de tener el puerto 8000 abierto en tu router (Port Forwarding) apuntando a tu PC.
  3. Edita el `config.json` y pon tu IP Pública: "http://TU_IP_PUBLICA:8000".

---------------------------------------------------------
PASO 4: ENVIAR A TUS AMIGOS
---------------------------------------------------------
1. Crea una carpeta nueva (ej: "Hermes_Para_Amigo").
2. Copia dentro el archivo `HermesV7.exe`.
3. Copia dentro el archivo `config.json` (ya editado con la IP correcta).
4. Comprime esa carpeta en un archivo ZIP (Click derecho -> Enviar a -> Carpeta comprimida).
5. Envía ese ZIP a tu amigo.

Tu amigo solo tiene que descomprimirlo y ejecutar `HermesV7.exe`.
